#  💳 FinTech: Motor de Reglas de Fraude

## Desafio: Evaluar transacciones entrantes. Si el monto > X o la ubicación es inusual, marcar como sospechosa.


## Visión del Producto

Desarrollar un sistema antifraude que permita detectar transacciones sospechosas de acuerdo al monto y ubicación. al ubicar montos superiores a un umbral o ubicaciones inusuales, se conseguirá aumentar la seguridad de las transacciones y reducir el riego de fraudes, garantizando una mejor experiencia del usuario.

## Objetivos
    1. Detectar transacciones potencialmente fraudulentas mediante reglas de negocio configurables. 

    2. Evaluar transacciones en tiempo real para identificar comportamientos anómalos como montos inusuales o ubicaciones atípicas. 

    3. Reducir el riesgo de fraude financiero mediante alertas tempranas sobre transacciones sospechosas. 

    4. Permitir que el sistema marque transacciones como sospechosas para que puedan ser revisadas o procesadas con mayor control. 

    5. Construir una base escalable que permita agregar nuevas reglas antifraude en el futuro.
